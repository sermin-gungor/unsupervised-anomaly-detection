# Social Media Anomaly Detection

Bu projede, öğrencilerin sosyal medya kullanım verileri üzerinde  
**One-Class SVM (OCSVM)** kullanılarak **model-tabanlı denetimsiz anomali (aykırı değer) tespiti** gerçekleştirilmiştir.

Proje kapsamında veri ön işleme, model eğitimi, doğrulama ve  
**PCA tabanlı görselleştirme** adımları uygulanmıştır.

---

## Kullanılan Yöntemler
- Model-based Anomaly Detection
- One-Class SVM (OCSVM)
- PCA tabanlı görselleştirme
- Performans analizi ve doğrulama
